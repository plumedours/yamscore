Politique de confidentialité — YamScore

Dernière mise à jour : 2025-10-16

YamScore respecte votre vie privée. L’application ne collecte, ne stocke, ni ne partage aucune donnée personnelle sur des serveurs externes. Toutes les informations sont enregistrées localement sur votre appareil.

Données traitées

  • Joueurs : noms, couleurs, avatar/emoji.
  • Parties : scores, historiques et statistiques dérivées.

Ces données sont stockées localement via une base de données embarquée (Hive). Aucune transmission à un service tiers n’est effectuée.

Connexion Internet

YamScore ne requiert aucune connexion Internet pour fonctionner.

Suppression des données

Depuis la page Paramètres → Zone de danger, vous pouvez effacer toutes les données (joueurs, parties, statistiques) enregistrées localement.

Partage

L’application ne partage aucune donnée automatiquement. Tout partage éventuel (ex. capture d’écran) est initié manuellement par l’utilisateur.

Enfants

YamScore peut être utilisé par tout public. Aucune donnée personnelle n’est collectée.

Contact

Pour toute question : via GitHub Issues sur le dépôt officiel.
